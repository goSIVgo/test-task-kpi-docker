1. **Docker Compose оркестрация:**

   - Вынос секретов в .env

2. **MySQL задачи:**

   - Наполнение тестовыми данными

## 🛠 Как запустить
1. Клонируйте репозиторий
2. Создайте файл `.env` на основе `.env.example`
3. Запустите контейнеры:
   ```bash
   docker-compose up -d
   ```

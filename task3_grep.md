# Задание 3: Поиск по содержимому файлов (Ubuntu Terminal)

## 🎯 Задача
Найти строку в файлах с помощью базовых команд терминала Ubuntu.

## ✅ Решение

### Основная команда:
```bash
grep -rn "искомый_текст" /путь/к/директории
```
✅ Флаги: `-r` (рекурсивно), `-n` (номер строки), `-i` (регистр)
✅ Фильтрация по типам файлов (`--include`)
